Part12.@extend 선택자 속성 가져오기

@extend : 클래스 이름을 전달하면
.btn { }
.order { @extend .btn } // 컴파일 된 파일에 .btn 클래스가 생김 @extend : %로 전달하면
%shape { }
.card-item { @extend %shape } // 컴파일된 파일에 shape 클래스가 생기지 않음
